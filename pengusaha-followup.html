Kolam Pengusaha

Pola copywriting elegan untuk pebisnis dengan pendekatan diversifikasi aset (Ban Serep).

Pola 1: Pendekatan Kasual (Sesama Pebisnis)

Gunakan script ini untuk relasi bisnis yang sudah Anda kenal cukup dekat.

Halo Pak/Bu [Nama], gimana perkembangan usahanya? Semoga makin cuan ya! Oh ya, belakangan ini saya lagi diversifikasi aset ke sebuah project digital nih. Sistemnya bagus banget buat jadi 'ban serep' atau dana darurat kalau bisnis lagi up and down, tanpa pusing mikirin operasional. Kalo lagi santai, kita ngopi yuk sambil saya kasih lihat konsepnya. Siapa tau cocok buat nambah portofolio Bapak/Ibu.

Pola 2: Pendekatan Formal / Direct (Cold Market)

Gunakan script ini untuk pemilik usaha yang baru Anda kenal secara profesional.

Selamat [pagi/siang] Pak/Bu [Nama]. Perkenalkan saya [Nama Anda]. Saya mengamati profil Bapak/Ibu sebagai pebisnis yang luar biasa. Saya saat ini sedang berekspansi mengedukasi para pengusaha mengenai instrumen diversifikasi aset untuk meminimalisir resiko bisnis konvensional dan mengamankan cashflow. Jika Bapak/Ibu berkenan, saya ingin berdiskusi singkat dan membagikan konsepnya. Mohon info waktu luangnya.
